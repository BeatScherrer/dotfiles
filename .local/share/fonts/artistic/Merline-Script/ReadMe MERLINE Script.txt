Access Glyph

If using a Mac, the easiest way to find the glyph you require is to use the Font Book application (included in the standard installation of OSX) 
Choose View Repertoire from the application menu and select the font to display all characters included. Click to select any of the displayed glyphs 
and copy / paste it back into Photoshop
